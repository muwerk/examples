A wireless display for MQTT values
==================================

A simple example demonstrating muwerk's scheduler, munet interface to WLAN, NTP, OTA and MQTT, with an OLED or TFT display that shows values of sensors via MQTT e.g. from Home-Assistant.

T.B.D.

## Building with platformio

* In data directory, copy `net-default.json` to `net.json` and `mqtt-default.json` to `mqtt.json`
* Customize `net.json` and `mqtt.json`
* Build filesystem: `pio run -t buildfs`
* Upload filesystem: `pio run -t uploadfs`
* Upload program code: `pio run -t upload`
